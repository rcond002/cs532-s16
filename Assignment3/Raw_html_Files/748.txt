class="ocpArticleContent" id="ocpArticleContent">\n<!--Last Published: 2/12/2016 8:33:20 AM PST-->\n<h1>What\'s new in Access 2016</h1>\n<section class="ocpIntroduction">\n<p>Access 2016 has all the functionality and features you\'re used to with some added enhancements and the best new features from Office 2016. \r\n\r\nHere are some of the top new features you\'ll find in Access 2016:\r\n</p>\n<p>\n<b class="ocpNote">\n<include item="note"></include> Note </b>\xc2\xa0\xc2\xa0For Access 2016, no existing features or functionality from previous versions were deprecated in this release.</p>\n</section>\n<section class="ocpSection">\n<h2>Upgrading from Access 2013</h2>\n<p>If you\'re upgrading to Access 2016 from Access 2013, you\'ll still have all the features you\'re used to\xe2\x80\x94and you\'ll notice a few new ones as well. The first thing you\xe2\x80\x99ll see when you open Access 2016 is that it has the familiar look of 2013 with a more colorful border.</p>\n</section>\n<section class="ocpSection">\n<h2>Do things quickly with Tell Me</h2>\n<p>You\'ll notice a text box on the ribbon in Access 2016 that says <b class="ocpUI">Tell me what you want to do</b>. This is a text field where you can enter words and phrases related to what you want to do next and quickly get to features you want to use or actions you want to perform. You can also choose to get help related to what you\'re looking for.</p>\n<p>Can\'t find a button? Click inside the Tell Me box (it\'s the box at the top, with the light bulb). Type a button or command, like "filter", and you\'ll see all of your filter-related options listed for you.</p>\n<p>\n<img alt=\'Shows the "tell me" search box on the ribbon in Access\' src="https://osiprodeusodcspstoa01.blob.core.windows.net/en-us/media/0585aa59-fc1d-44a1-8c99-48677c488897.png" title=\'Shows the "tell me" search box on the ribbon in Access\'/>\n</p>\n<p>\n<img alt="Shows a query being entered into the tell me field on the ribbon in Access" src="https://osiprodeusodcspstoa01.blob.core.windows.net/en-us/media/3b89f5c4-ec0d-40c0-9399-61113088b3f8.png" title="Shows a query being entered into the tell me field on the ribbon in Access"/>\n</p>\n<p>\n<b class="ocpNote">\n<include item="note"></include> Note </b>\xc2\xa0\xc2\xa0The Tell Me box is not available in the ribbon when designing Access web apps.</p>\n</section>\n<section class="ocpSection">\n<h2>New themes for the Access program</h2>\n<p>There are now two Office themes that you can apply to the Access program: Colorful and White. To access these themes, go to <b class="ocpUI">File</b> &gt; <b class="ocpUI">Options</b> &gt; <b class="ocpUI">General</b>, and then click the drop down menu next to <b class="ocpUI">Office Theme</b>.</p>\n<p>\n<img alt="Office Theme pull down menu, Colorful and White theme options" src="https://osiprodeusodcspstoa01.blob.core.windows.net/en-us/media/be35bde9-a6a2-4182-966e-f1da5791eb6d.png" title="Office Theme pull down menu, Colorful and White theme options"/>\n</p>\n</section>\n<section class="ocpSection">\n<h2>Export linked data source information to Excel</h2>\n<p>Have you ever wanted to get a nice list of all the linked data sources from your Access database application into Excel? If you are working on a complex Access application, for example, that includes links to many different data sources, it can be helpful to have a nice list of all the various data sources and their types. This exported list can be especially helpful if you are working on an Access application you did not originally design. Now with Access 2016, you\'ll find this task much easier using new functionality built into the Linked Table Manager dialog.</p>\n<p>Open the Linked Table Manager dialog by clicking <b class="ocpUI">External Data</b> &gt; <b class="ocpUI">Linked Table Manager</b>. Select the linked data sources you want to list and then click <b class="ocpUI">Export to Excel</b>.</p>\n<p>\n<img alt="The linked table manager dialog in Access with the Export to Excel button selected." src="https://osiprodeusodcspstoa01.blob.core.windows.net/en-us/media/f5489e21-5dbf-4788-b4c3-2b5fac71097b.png" title="The linked table manager dialog in Access with the Export to Excel button selected."/>\n</p>\n<p>Access prompts you for a location to save the Excel workbook. After you provide a save location, Access displays the linked data sources information in the new workbook. You\'ll see the name of the linked data source, its source information, and the data source type.</p>\n<p>\n<img alt="Excel workbook displaying linked table information from Access" src="https://osiprodeusodcspstoa01.blob.core.windows.net/en-us/media/2f60704a-b19b-47f8-9f87-53b368823755.png" title="Excel workbook displaying linked table information from Access"/>\n</p>\n<p>\n<b class="ocpNote">\n<include item="note"></include> Note </b>\xc2\xa0\xc2\xa0The Linked Table Manager dialog is not available when designing Access web apps.</p>\n</section>\n<section class="ocpSection">\n<h2>Templates get a modern look and feel</h2>\n<p>If you want to organize and manage your data with Access but you\'d like some help getting started with designing the database, try using a desktop database template. In Access 2016, five of the most popular database templates have been redesigned to have a more modern look and feel. To try these templates out yourself, go to <b class="ocpUI">File</b> &gt; <b class="ocpUI">New</b>, and then search for <b class="ocpUI">Asset tracking</b>, <b class="ocpUI">Contacts</b>, <b
