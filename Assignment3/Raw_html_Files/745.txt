class="ocpArticleContent" id="ocpArticleContent">\n<!--Last Published: 2/12/2016 7:40:22 AM PST-->\n<h1>What\'s new in Excel 2016 for Windows</h1>\n<section class="ocpIntroduction">\n<p>Excel 2016 for Windows has all the functionality and features you\'re used to, with some added features and enhancements. Here are some of the top new and improved features for Excel 2016.</p>\n</section>\n<section class="ocpSection">\n<h2>January 2016 Updates</h2>\n<p>\n<b class="ocpNote">\n<include item="note"></include> Note </b>\xc2\xa0\xc2\xa0The following features are only available to <a class="ocpExternalLink" href="https://products.office.com/en-us/buy/compare-microsoft-office-products" target="_blank" title="Office 365 subscribers">Office 365 subscribers</a>. They will first roll out to <a class="ocpExternalLink" href="https://office.com/insider" target="_blank" title="Office Insider">Office Insider</a> participants, and later to consumer and commercial Office 365 subscribers. If you have an Office 365 subscription, <a class="ocpExternalLink" href="https://support.office.com/article/ee68f6cf-422f-464a-82ec-385f65391350" target="_blank" title="make sure you have the latest version of Office">make sure you have the latest version of Office</a>.</p>\n<section class="ocpSection">\n<h3>Funnel charts</h3>\n<p>Funnel charts show values across multiple stages in a process. Typically, the values decrease gradually, allowing the bars to resemble a funnel. <a class="ocpArticleLink" href="Read-more-about-funnel-charts-here-ba21bcba-f325-4d9f-93df-97074589a70e" title="Read more about funnel charts here">Read more about funnel charts here</a>.</p>\n<p>\n<img alt="Funnel chart showing sales pipeline; stages listed in the first column, values in the second" src="https://osiprodeusodcspstoa01.blob.core.windows.net/en-us/media/972f3492-4573-485e-91df-0c51aa84ed99.png" title="Funnel chart showing sales pipeline; stages listed in the first column, values in the second"/>\n</p>\n</section>\n<section class="ocpSection">\n<h3>Improved autocomplete</h3>\n<p>Excel autocomplete isn\'t as picky as it was before. For example, let\'s say you want to use the  NETWORKDAYS function, but you can\'t remember how it is spelled. If you just type <b class="ocpLiteral">=DAYS</b>, the autocomplete menu will bring back all of the functions that contain "DAYS," including, <b class="ocpUI">NETWORKDAYS</b>. (Before, you had to spell the function name exactly.) </p>\n</section>\n<section class="ocpSection">\n<h3>TEXTJOIN</h3>\n<p>This function combines text from multiple ranges, and each item is separated by a delimiter that you specify. <a class="ocpArticleLink" href="Learn-more-about-TEXTJOIN-here-357b449a-ec91-49d0-80c3-0e8fc845691c" title="Learn more about TEXTJOIN here">Learn more about TEXTJOIN here</a>.</p>\n</section>\n<section class="ocpSection">\n<h3>CONCAT</h3>\n<p> This new function is like CONCATENATE, but better. First of all: it\'s shorter and easier to type. But it also supports range references in addition to cell references.  <a class="ocpArticleLink" href="Learn-more-here-9b1a9a3f-94ff-41af-9736-694cbd6b4ca2" title="Learn more here">Learn more here</a>.</p>\n</section>\n<section class="ocpSection">\n<h3>IFS</h3>\n<p> Tired of typing complicated, nested IF functions? The IFS function is the solution. With this function, conditions are tested in the order that you specify. If passed, the result is returned. You can also specify an else "catch all" if none of the conditions are met. <a class="ocpArticleLink" href="Learn-about-IFS-here-36329a26-37b2-467c-972b-4a39bd951d45" title="Learn about IFS here">Learn about IFS here</a>.</p>\n</section>\n<section class="ocpSection">\n<h3>SWITCH</h3>\n<p> This function evaluates an expression against a list of values in order, and returns the first matching result. If no results match, the "else" is returned. \r\n<a class="ocpArticleLink" href="Details-and-examples-are-spelled-out-here-47ab33c0-28ce-4530-8a45-d532ec4aa25e" title="Details and examples are spelled out here">Details and examples are spelled out here</a>.</p>\n</section>\n<section class="ocpSection">\n<h3>MAXIFS</h3>\n<p> This function returns the largest number in a range, that meets a single or multiple criteria. <a class="ocpArticleLink" href="More-information-available-here-dfd611e6-da2c-488a-919b-9b6376b28883" title="More information available here">More information available here</a>.</p>\n</section>\n<section class="ocpSection">\n<h3>MINIFS</h3>\n<p>This function is similar to MAXIFS, but it returns the smallest number in a range, that meets a single or multiple criteria. <a class="ocpArticleLink" href="Read-more-about-the-MINIFS-function-6ca1ddaa-079b-4e74-80cc-72eef32e6599" title="Read more about the MINIFS function">Read more about the MINIFS function</a>.</p>\n</section>\n</section>\n<section class="ocpSection">\n<h2>September 2015</h2>\n<p>Unless otherwise specified, both Office 365 subscribers and Office 2016 one-time purchasers get the following new features.</p>\n<section class="ocpSection">\n<h3>Six new chart types</h3>\n<p>Visualizations are critical to effective data analysis as well as compelling storytelling. In Excel 2016, we\'ve added six new charts\xe2\x80\x94with the same rich formatting options that you are familiar with\xe2\x80\x94to help you create some of the most commonly used data visualizations of financial or hierarchal information or for revealing statistical properties in your data.</p>\n<p>Click <b class="ocpUI">Insert Hierarchy Chart</b> on the<b class="ocpUI"> Insert</b> tab to use the <b class="ocpUI">Treemap</b> or <b class="ocpUI">Sunburst</b> chart, click <b class="ocpUI">Insert Waterfall or Stock Chart</b> for <b class="ocpUI">Waterfall</b>, or click <b class="ocpUI">Insert Statistical Chart</b> for <b class="ocpUI">Histogram</b>, <b class="ocpUI">Pareto</b>, or <b class="ocpUI">Box and Whisker</b>.</p>\n<p>\n<img alt="Icons for inserting hierarchy, waterfall or stock, or statistical charts in Excel 2016 for Windows" src="https://osiprodeusodcspstoa01.blob.core.windows.net/en-us/media/ad006678-b459-4e8b-afc6-ec9ed8c0a4ff.png" title="Icons for inserting hierarchy, waterfall or stock, or statistical charts in Excel 2016 for Windows"/>\n</p>\n<p>Or  click <b class="ocpUI">Recommended Charts</b> &gt; <b class="ocpUI">All Charts</b> to see all the new charts.</p>\n<p>\n<img alt="shows the new charts in PowerPoint2016" src="https://osiprodeusodcspstoa01.blob.core.windows.net/en-us/media/d775594a-72d2-4c1e-a807-3a8fd9ed2323.png" title="shows the new charts in PowerPoint2016"/>\n</p>\n</section>\n<section class="ocpSection">\n<h3>Get and transform (Query)</h3>\n<p>Before analysis can begin, you must be able to bring in the data relevant to the business question you are trying to answer. Excel 2016 now comes with built-in functionality that brings ease and speed to getting and transforming your data\xe2\x80\x94allowing you to find and bring all the data you need into one place. These new capabilities, previously only available as a separate add-in called Power Query, can be found natively within Excel. Access them from the <b class="ocpUI">Get &amp; Transform</b> group on the <b class="ocpUI">Data</b> tab.</p>\n<img alt="Data sources for Query" src="https://osiprodeusodcspstoa01.blob.core.windows.net/en-us/media/19e2c94c-ac9b-451b-bd33-1047912f58a0.png" title="Data sources for Query"/>\n</section>\n<section class="ocpSection">\n<h3>One click forecasting</h3>\n<p>In previous versions of Excel, only linear forecasting had been available. In Excel 2016, the FORECAST function has been extended to allow forecasting based on Exponential Smoothing (such as, FORECAST.ETS() \xe2\x80\xa6). This functionality is also available as a new one-click forecasting button. On the <b class="ocpUI">Data</b> tab, click the <b class="ocpUI">Forecast Sheet</b> button to quickly create a forecast visualization of your data series. From the wizard, you can also find options to adjust common forecast parameters, like seasonality, which is automatically detected by default and confidence intervals.</p>\n<img alt="Part of a spreadsheet showing the table of forecasted numbers and a forecast chart" src="https://osiprodeusodcspstoa01.blob.core.windows.net/en-us/media/bf52dcc4-3a1e-4db3-bf2c-2f796d01f01e.png" title="Part of a spreadsheet showing the table of forecasted numbers and a forecast chart"/>\n</section>\n<section class="ocpSection">\n<h3>3D Maps</h3>\n<p>Our popular 3D geospatial visualization tool, Power Map, has been renamed and is now available to all Excel 2016 customers and is built into Excel. This innovative set of storytelling capabilities has been renamed 3D Maps and can be found along with other visualization tools by clicking <b class="ocpUI">3D Map</b> on the  <b class="ocpUI">Insert</b> tab.</p>\n<p>\n<img alt="Power Map tour set to replay" src="https://osiprodeusodcspstoa01.blob.core.windows.net/en-us/media/0986e974-59f3-4c38-b5a1-79eebcca60df.png" title="Power Map tour set to replay"/>\n</p>\n</section>\n<section class="ocpSection">\n<h3>Financial templates</h3>\n<p>Take advantage of \tthe new  <a class="ocpArticleLink" href="My-Cashflow-template-215e9e2e-5813-41ad-a9ef-a0c0874841bb" title="My Cashflow template">My Cashflow template</a>, and the <a class="ocpArticleLink" href="Stock-Analysis-template-f65e62ac-7af6-4cc6-98f3-f68b147ed65d" title="Stock Analysis template">Stock Analysis template</a>. These templates track what  you earn, how much you spend, and where your spending occurs. Plus, quickly analyze and compare the performance of selected stocks over time.</p>\n</section>\n<section class="ocpSection">\n<h3>Calendar Insights Template</h3>\n<p>\n<a class="ocpArticleLink" href="View-your-calendar-as-a-dashboard-7edbeb88-99ca-403f-a394-7e957d3d3f40" title="View your calendar as a dashboard">View your calendar as a dashboard</a> and drill into the data. You\'ll get a better handle on how you spend your time, and identify ways to get more out of your days.</p>\n</section>\n<section class="ocpSection">\n<h3>PivotTable enhancements</h3>\n<p>Excel is known for its flexible and powerful analysis experiences, through the familiar PivotTable authoring environment. With Excel 2010 and Excel 2013, this experience was significantly enhanced with the introduction of Power Pivot and the Data Model, bringing the ability to easily build sophisticated models across your data, augment them with measures and KPIs, and then calculate over millions of rows with high speed. Here are some of the enhancements we made in Excel 2016,  so that you can focus less on managing your data and more on uncovering the insights that matter.\r\n</p>\n<ul>\n<li>\n<p>\n<b class="ocpRunInHead">Automatic relationship detection</b>\xc2\xa0\xc2\xa0\xc2\xa0 discovers and creates relationships among the tables used for your workbook\xe2\x80\x99s data model, so you don\xe2\x80\x99t have to. Excel 2016 knows when your analysis requires two or more tables to be linked together and notifies you. With one click, it does the work to build the relationships, so you can take advantage of them immediately.</p>\n</li>\n<li>\n<p>\n<b class="ocpRunInHead">Creating, editing and deleting custom measures</b>\xc2\xa0\xc2\xa0\xc2\xa0 can now be done directly from the PivotTable fields list, saving you a lot of time when you need to add additional calculations for your analysis.</p>\n</li>\n<li>\n<p>\n<b class="ocpRunInHead">Automatic time grouping</b>\xc2\xa0\xc2\xa0\xc2\xa0 helps you to use your time-related fields (year, quarter, month) in your PivotTable more powerfully, by auto-detecting and grouping them on your behalf. Once grouped together, simply drag the group to your PivotTable in one action and immediately begin your analysis across the different levels of time with drill-down capabilities.</p>\n<p>\n<b class="ocpRunInHead">PivotChart drill-down buttons</b>\xc2\xa0\xc2\xa0\xc2\xa0 allow you to zoom in and out across groupings of time and other hierarchical structures within your data.</p>\n</li>\n<li>\n<p>\n<b class="ocpRunInHead">Search in the PivotTable</b>\xc2\xa0\xc2\xa0\xc2\xa0 field list helps you get to the fields that are important to you across your entire data set.</p>\n</li>\n<li>\n<p>\n<b class="ocpRunInHead">Smart rename</b>\xc2\xa0\xc2\xa0\xc2\xa0 gives you the ability to rename tables and columns in your workbook\xe2\x80\x99s data model. With each change, Excel 2016 automatically updates any related tables and calculations across your workbook, including all worksheets and DAX formulas.</p>\n</li>\n<li>\n<p>\n<b class="ocpRunInHead">Multiple usability improvements</b>\xc2\xa0\xc2\xa0\xc2\xa0 have also been made. For example, delayed updating allows you to perform multiple changes in Power Pivot without the need to wait until each is propagated across the workbook. The changes will be propagated at one time, once the Power Pivot window is closed.</p>\n</li>\n</ul>\n<img alt="PivotTable with callouts showing the new features in Excel 2016" src="https://osiprodeusodcspstoa01.blob.core.windows.net/en-us/media/3c61fedf-3380-4871-8c33-2f916423fcb5.png" title="PivotTable with callouts showing the new features in Excel 2016"/>\n</section>\n<section class="ocpSection">\n<h3>Multi-select slicer</h3>\n<p>Now you can select multiple items in an Excel slicer on a touch device.
